표 만들기

HTML에서 표를 구현할 때 사용하는 태그

  • tr(table row):행
  • td(table data):열(행의 내용)
  • th(table heading):행,열의 제목(굵은체,가운데 정렬)
  • 속성 colspan="2" : 2칸의 열(column)을 병합
  • 속성 rowspan="3" : 3칸의 행(row)을 병합
